# Actions PoC

To run anything remember to **IGNORE** directory and file extension.

## Setup:

```
yarn install
```

## Development:

```
yarn dev index
yarn dev usage
yarn dev guess-the-number
```

## Build

```
yarn build index
yarn build usage
yarn build guess-the-number
```

## Execute

```
yarn start index
yarn start usage
yarn start guess-the-number
```
